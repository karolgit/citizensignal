# Wireframes

This directory contains a draw.io prototype for two primary user types:

- administrator
- campaign manager

## Files

- `citizensignal-admin-campaign-prototype.drawio`: responsive wireframe prototype with desktop and mobile pages
- `clickable-prototype.html`: clickable browser prototype
- `prototype-styles.css`: styling for the clickable prototype
- `prototype-app.js`: local interaction and screen rendering logic
- `prototype-sample-data.json`: sample content used by the prototype

## Prototype Pages

- `Administrator Desktop`
- `Campaign Manager Desktop`
- `Administrator Mobile`
- `Campaign Manager Mobile`
- `Login`
- `District Setup`
- `Sources`

## Design Direction

The mockups use a Liquid Glass inspired visual system:

- translucent layered cards
- soft blue-white gradients
- restrained chrome and strong content focus
- consistent navigation patterns across desktop and mobile

## Responsive Intent

The desktop pages use a sidebar plus multi-column card layout.

The mobile pages collapse the same content into:

- a single-column card stack
- simplified top navigation
- action-first summaries for small screens

Open the `.drawio` file in diagrams.net or draw.io to edit the prototype.

Open `clickable-prototype.html` directly in a browser to use the clickable prototype.
